# Dotfiles

Personal dotfiles for macOS development environment.

## Quick Setup

```bash
git clone https://github.com/yourusername/dotfiles.git ~/Developer/dotfiles
cd ~/Developer/dotfiles/os
./install.sh
```

See [os/README.md](os/README.md) for detailed setup instructions.

## What's Included

- **nvim** - Neovim configuration
- **tmux** - Terminal multiplexer
- **zsh** - Shell configuration
- **starship** - Cross-shell prompt
- **ghostty** - Terminal emulator config
- **obsidian** - Vim keybindings
- **claude** - Claude Code custom skills
- **os** - macOS setup automation (Brewfile, install scripts)
